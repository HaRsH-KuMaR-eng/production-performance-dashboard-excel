# 📊 Production Performance Dashboard (Excel)

An interactive **Production Dashboard** built in **Microsoft Excel** using Pivot Tables, Pivot Charts, Slicers, and Dashboard Design techniques. This project provides a comprehensive overview of production performance, manufacturing costs, units produced, manager productivity, and product-wise cost analysis.

# 📌 Project Overview

This dashboard helps manufacturing companies monitor production activities by analyzing:

- Total Production Cost
- Production Cost per Unit
- Total Units Produced
- Product-wise Performance
- Manager-wise Task Distribution
- Monthly Production Trends

The dashboard is fully interactive through slicers, allowing users to filter data by:

- Region
- Gender
- Age Group
- Production Quarter

---

# 🎯 Business Problem

Manufacturing companies generate thousands of production records every month. Analyzing these records manually is time-consuming and makes it difficult to identify:

- High production cost products
- Monthly production trends
- Manager workload distribution
- Average production cost per unit
- Regional production performance

This dashboard provides a centralized solution for monitoring production KPIs and making data-driven decisions.

# 📈 Dashboard KPIs

### 1️⃣ Total Production Cost by Product Type

Displays the overall production cost for each product category.

**Insights**

- Automobiles have the highest production cost.
- Electronics have the lowest production cost.
- Machinery is the second-highest cost category.

---

### 2️⃣ Number of Tasks by Manager

Compares workload among managers.

**Insights**

- Nancy Grey completed the highest number of tasks.
- Jane Smith ranks second.
- Sarah Lee handled the fewest tasks.

---

### 3️⃣ Monthly Units Produced

Tracks production trends over time.

**Insights**

- Production peaked in November 2023.
- A noticeable decline occurred during April–May 2024.
- Production recovered in June before dropping again.

---

### 4️⃣ Average Production Cost per Unit

Shows average manufacturing cost across product categories.

**Insights**

- Furniture has the highest average production cost per unit.
- Electronics and Machinery have comparatively lower production costs.
